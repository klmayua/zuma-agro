- **All spacing values must be multiples of 8px**

## Spacing Rules

1. **8px Grid:** All spacing must be multiples of 8px
2. **Consistency:** Use tokens, not arbitrary values
3. **Responsive:** Reduce spacing on smaller screens
4. **Vertical Rhythm:** Maintain consistent vertical spacing
5. **Whitespace:** Use generous whitespace for institutional feel
